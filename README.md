# How to install
1. Put this into your terminal (Linux):

        cd /var/www/yourvichandirectoryname
        composer install
2. Then do this;

        chown -R www-data:www-data /var/www/yourvichandirectoryname
4. Set up the database (MySQL only) and, then go to the /install.php directory on your URL, and set it up.
